## Experiment 001: Reducing System Animation Lag

### Objective
Test whether reducing Android system animation scales improves
perceived performance on low-end devices.

### Method
Change the following system settings:
- window_animation_scale
- transition_animation_scale
- animator_duration_scale

### Expected Outcome
Faster UI response and reduced lag when navigating the system.

### Notes
This experiment uses standard Android settings and does not
require root access.
